# job4j_tracker

## About the Project

![CI GitHubAction](https://github.com/peterarsentev/job4j_tracker/actions/workflows/maven.yml/badge.svg)

This project is a console application. The user is presented with a menu showing the program's features.

The program can:

1. Add a new item.

2. Replace an item with a new one by ID.

3. Delete an item by ID.

4. Display a list of all items.

5. Search for an item by name.
